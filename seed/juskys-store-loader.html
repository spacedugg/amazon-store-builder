Juskys Store hochladen

Lädt das fertige Store JSON in deine laufende Vercel App. Kein Terminal nötig.

Wird geladen...
  1. URL oben eintragen, ohne Pfad. Beispiel: https://amazon-store-builder.vercel.app
  2. Auf Store hochladen klicken. Du bekommst danach einen Share Link.
  3. Im offenen Builder Tab links auf + Import Store klicken, den Share Link einfügen, Enter.
